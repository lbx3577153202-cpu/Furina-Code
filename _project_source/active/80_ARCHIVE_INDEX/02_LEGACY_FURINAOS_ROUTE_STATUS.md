# 旧 FurinaOS 路线状态

旧 FurinaOS / Local Autopilot 仓库和资源包是 Furina Code 的历史来源，不是新 `Furina-Code` 仓库的直接代码基线。

## 保留价值

- 用户低打扰原则；
- 任务卡和治理经验；
- Gate、Evidence、Recovery、Memory 的早期试验；
- preview/dry-run 被过度声明的失败经验；
- 造物台和开发方式的概念来源。

## 当前无权威的内容

- 旧 HEAD、分支、测试数量；
- 旧器官成熟度；
- 旧产品入口和 UI；
- 旧 Limited Source Write 能力；
- 旧目录映射；
- 将 Furina Code 定义为旧系统中的一个 profile。

旧材料必须通过 `60_PROVENANCE/05_LEGACY_TO_CURRENT_TRANSLATION.md` 进入当前讨论。
